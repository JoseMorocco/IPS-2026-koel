Proyecto

¿Por qué Koel?

Seleccionamos Koel como base del proyecto por cumplir todos los criterios técnicos y académicos requeridos: licencia MIT, stack moderno, complejidad real y soporte Docker nativo.

Objetivo Académico

Implementar un proceso de desarrollo de software que integre la metodología ágil Scrum con prácticas DevOps automatizadas, aplicado sobre Koel, un proyecto open source con licencia MIT.

El equipo analizará el estado actual del proyecto, identificará funcionalidades y arquitectura, definirá un alcance de mejora y aplicará un modelo de proceso que integre Scrum con buenas prácticas de automatización CI/CD.

◈ Frontend — Vue.js

SPA con componentes como App.vue, MainWrapper, AudioPlayer, AppFooter y menús contextuales. Gestión de estado con songStore, albumStore, artistStore, queueStore y playbackService.

◈ Backend — Laravel

Gestiona lógica de negocio, autenticación, rutas API y procesamiento de metadatos. Expone endpoints para canciones, álbumes y artistas, y maneja colas de trabajo complejas.

◈ Media & Database

Archivos multimedia en local o Dropbox. La BD almacena usuarios, canciones, artistas, álbumes, listas de reproducción y favoritos. Compatible con MySQL, MariaDB, PostgreSQL y SQLite.
